# Web Vulnerability Assessment Report

## Website Tested
http://testasp.vulnweb.com

## Scope
This assessment was limited to passive scanning only. No exploitation was performed.

## Tools Used
- OWASP ZAP
- Nmap
- Browser Developer Tools

## Methodology
1. Performed Nmap scan
2. Conducted OWASP ZAP passive scan
3. Verified findings using DevTools

## Files Included
- Report.pdf
- Evidence (Screenshots)
